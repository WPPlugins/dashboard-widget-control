== Description ==

Enable and Disable the standard WordPress Dashboard Widgets without any code modification.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Disable WordPress Dashboards Plugin Menu to enable/disable Dashboard widgets.
